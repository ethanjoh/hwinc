# 변경 이력 (Changelog)

이 프로젝트의 업데이트 및 최적화 작업 내역을 날짜별로 정리한 변경 이력 파일입니다.

## [2026-06-26]

### 추가 및 개선 사항 (Feat & Refactor)
- **메인 배경 동영상 최적화 및 교체 (`home_movie5`)**
  - 메인 페이지(`index.php`)의 배경 비디오를 새 리소스인 `home_movie5.mp4`로 업데이트했습니다.
  - **대역폭 최적화**: 무음(Mute) 배경 영상임을 감안해 오디오 트랙을 완전히 삭제하여 파일 크기를 크게 단축했습니다.
  - **MP4 용량 감소**: H.264 코덱 및 적정 품질 수준(CRF 30)을 사용해 용량을 기존 **2.56 MB**에서 **1.34 MB (약 47.7% 절감)**로 축소했습니다.
  - **WebM 지원 추가**: 최신 웹 환경 및 모바일에 최적화된 고압축 WebM 포맷(`home_movie5.webm`, **1.50 MB, 약 41.4% 절감**)을 신규 인코딩해 배치했습니다.
  - **스트리밍 최적화 (Faststart)**: 다운로드가 100% 완료되기 전에 영상이 브라우저에서 즉시 스트리밍 및 재생될 수 있도록 MP4 메타데이터(`moov atom`) 위치를 파일 시작 부분으로 재배치했습니다.
  - 기존 배경 영상인 `home_movie3.mp4` 및 WebM 버전에 대해서도 동일 규격으로 최적화 완료했습니다.
  - 원본 영상 파일들은 접미사 `_bak`를 붙여(`home_movie5_bak.mp4` 등) 안전하게 백업을 구성했습니다.
- **사업분야 동영상 최적화 및 멀티 소스 적용 (`shred.php`, `recycle.php`)**
  - **대상 영상**: 문서파쇄 페이지의 `shredding2.mp4` 및 폐동케이블 재활용 페이지의 `cable2.mp4` 동영상
  - **대역폭 최적화**: 무음 자동 재생 비디오 특성에 맞춰 오디오 트랙을 제거하여 파일 크기를 크게 절감했습니다.
  - **용량 최적화**:
    - `shredding2.mp4` 용량을 기존 **2.70 MB**에서 **1.11 MB (H.264, 약 58% 절감)**로 축소하고 **1.12 MB의 WebM** 버전도 신규 생성하여 최적 배치했습니다.
    - `cable2.mp4` 용량을 기존 **1.61 MB**에서 **597 KB (H.264, 약 64% 절감)**로 축소하고 **495 KB의 WebM** 버전도 신규 생성하여 최적 배치했습니다.
  - **스트리밍 & 체감 로딩 속도 극대화**:
    - 첫 프레임을 추출한 스크린샷 이미지(`shredding2_poster.jpg`, `cable2_poster.jpg`)를 포스터(`poster`) 속성으로 지정하여 즉각적인 로딩 화면을 보여주도록 구성했습니다.
    - MP4 메타데이터 정렬(`-movflags +faststart`)을 마쳐 스트리밍 재생이 끊김 없이 신속하게 시작되도록 개선했습니다.
  - **멀티 소스 지원**: `<video>` 태그 내에 고효율 `.webm` 파일을 우선 탑재하고 하위 호환성을 위해 `.mp4`로 폴백하는 방식으로 태그를 보완했습니다.
  - 원본 영상들은 `_orig` 접미사를 붙여 백업하였습니다.

## [2026-06-25]

### 기능 추가 및 개선 (Feat & Refactor)
- **메인 페이지 히어로 영역 비디오 배너 전환**
  - 기존 정적 배너 이미지 노출 방식에서 자동 재생되는 HTML5 비디오 레이어로 교체했습니다.
  - Revolution Slider 모듈 내에서 비디오 로드 로직을 구성했습니다.
  - 사용되지 않는 두 번째 슬라이드 코드를 주석 처리하여 불필요한 스크립트 실행 및 네트워크 부하를 해소했습니다.
- **파쇄 프로세스 페이지 영상 리소스 탑재 (`shred.php`)**
  - 문서 파쇄 시연 과정을 보여주는 전용 비디오 에셋을 추가하고, 페이지 내 재생 로직을 구현했습니다.
- **재활용 페이지 동영상 레이아웃 개선 (`recycle.php`)**
  - 공정 시연 동영상 요소를 추가로 배치했습니다.
  - 설명 문단(`p` 태그)들의 마크업을 구조화하여 페이지 전체 레이아웃의 정렬 상태 및 가독성을 보완했습니다.
- **회사소개 및 위치 레이아웃 조정**
  - 회사소개 페이지(`about-us.php`)의 이미지 스케일과 문단 간 여백(마진/패딩)을 화면 규격에 맞춰 미세 조정했습니다.
  - 지도 영역 안내 팝업(인포윈도우)의 설명 텍스트 디자인과 레이아웃 구조를 깔끔하게 보정했습니다.
